landing-m2
landing-m2 — это прямое продолжение проекта landing-m1.
Цель проекта добавление Адаптивного Дизайна и Тёмной Темы.

НОВОЕ
Адаптивный дизайн
Интерфейс автоматически подстраивается под разные размеры экранов.
Существующие Breakpoint'ы: Мобильный: 375-599 пикселей.
Средний размер: 600-1439 пикселей.
Рабочий стол: 1440 пикселей и выше.
Темная тема
Поддержка светлой и тёмной темы с простым переключателем.
При нажатии на солнышко в правом верхнем углу тема сменится.
Сделано полностью на HTML и CSS, без JS.
Основные возможности
Главная страница: Приветственная секция с навигационными ссылками.
Страница проектов: Отображение различных проектов с возможностью фильтрации.
Используемые технологии
HTML5: Для создания структуры веб-страниц.
CSS3: Для стилизации и верстки, включая Flexbox для адаптивного дизайна.
Google Fonts: Для шрифтов.
w3c Validator: Для валидации html и css
Figma: Для примера готового дизайна
Инструкции по настройке
Клонируйте репозиторий:

git clone git@git.platform.alem.school:bekulymba/landing-m2.git
Перейдите в Каталог проекта:

cd landing-m2
Откройте файл index.html

Разбор сайта
Сайт соответсвует Semantic HTML.

Header: Содержит логотип и навигацию для удобства перехода по сайту.

Hero: Главный визуальный элемент, привлекающий внимание с основным предложением или информацией.

Features: Выделяет ключевые особенности или преимущества продукта/услуги.

Testimonial: Отзывы клиентов, повышающие доверие к бренду.

Blog: Статьи, делящиеся новыми идеями и информацией с пользователями.

FAQ: Раздел с ответами на часто задаваемые вопросы.

Contact: Контактная информация и форма для связи с компанией.

Footer: Нижняя часть сайта с дополнительной информацией, ссылками и контактами.
